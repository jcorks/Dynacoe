GUI Class Reference

#include <GUI.h>

Description

A component that provides managed input interaction suitable for making things like buttons, text boxes, etc.

Supported events:

  • "on-click"
  • "on-enter"
  • "on-leave"
  • "on-drag"
  • "on-focus"
  • "on-unfocus"
